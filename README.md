# demo  
个人博客的demo，以及其他情况下使用到demo  

## 使用方法

### 启动服务，查看demo

根目录下启动  
```
http-server ./  
```
为啥带 ./ ，我也不知道为啥根目录下启动http-server，他会默认到./public目录下，估带./当前目录。
访问对应端口即可查看demo。  

### 重新编译
```
npm run build
```
